# Block_Placement_Mechanic

A 3D building placement system made in Unity.

## 🚀 Features
- Grid-based placement (1 unit snapping)
- Cube stacking system
- Rotation system (R key)
- Placement preview system
- Camera movement (A/D keys)
- Placement mode toggle UI

## 🎮 Controls
- Left Click → Place cube
- R → Rotate cube
- A/D → Move camera
- Button → Toggle placement mode

## 🛠️ Tech Used
- Unity (C#)
- Raycasting
- Prefabs
- UI System
